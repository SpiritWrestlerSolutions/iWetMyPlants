# Code of Conduct

## The Short Version

Be kind. Be constructive. Assume good faith.

This is a hobby project maintained by one person. Everyone here — whether they're filing their first bug report or submitting their tenth pull request — deserves to be treated with respect.

## What This Looks Like in Practice

**We welcome:**
- Questions, even basic ones
- Bug reports, even incomplete ones
- Opinions that differ from the current design
- Criticism that's honest and specific
- People who are new to ESP32, to open source, or to electronics

**We don't welcome:**
- Dismissiveness toward beginners
- Personal attacks or insults
- Harassment of any kind
- Entitlement ("why haven't you fixed this yet?")
- Spam or self-promotion unrelated to the project

## Enforcement

If someone is behaving badly, please reach out privately via the contact information on the installer page. Issues that violate this code of conduct may be closed or removed at the maintainer's discretion.
