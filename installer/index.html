Flash New Device

Connect your ESP32 via USB and click Connect & Install. This will erase the board and install fresh firmware. Requires Chrome or Edge on desktop.

  1. Click Connect & Install and select your Hub's USB port
  2. Wait ~30 seconds for firmware to flash and the Hub to reboot
  3. A Configure WiFi dialog will appear вЂ” enter your home network SSID and password
  4. The Hub connects and shows an Open Device link to its web interface
  1. Click Connect & Install and select your Remote's USB port
  2. Wait ~30 seconds for firmware to flash and the Remote to reboot
  3. A Configure WiFi dialog will appear — enter your home network SSID and password
  4. The Remote connects and shows an Open Device link to its web interface
  1. Click Connect & Install and select your Greenhouse's USB port
  2. Wait ~30 seconds for firmware to flash and the Greenhouse to reboot
  3. A Configure WiFi dialog will appear — enter your home network SSID and password
  4. The Greenhouse connects and shows an Open Device link to its web interface
Web Serial is not supported in this browser. Please use Chrome or Edge (desktop) to flash via USB.

No WiFi dialog? Connect another device to the IWMP-Hub-XXXXXX WiFi network and visit http://192.168.4.1 to configure manually.

After flashing, the Remote creates a WiFi network named IWMP-Remote-XXXXXX. Connect to it from your phone or computer, then visit http://192.168.4.1 to enter your home WiFi credentials.

After flashing, the Greenhouse creates a WiFi network named IWMP-GH-XXXXXX. Connect to it from your phone or computer, then visit http://192.168.4.1 to enter your home WiFi credentials.

or update an existing device

Update via OTA

For devices already running iWetMyPlants firmware and connected to your network. Enter the device IP, select a firmware file, then click Update Firmware.

Note: Your browser will send this request directly to the device on your local network. The device firmware must serve Access-Control-Allow-Origin: * headers on /api/system/ota for this to work from an external page. If it fails with a CORS error, use the device's own Settings page (at its local IP) to upload firmware instead.
Find the IP on your router's DHCP list, or check the Hub's Devices page.
Leave blank to use the latest release build for the selected device type.
