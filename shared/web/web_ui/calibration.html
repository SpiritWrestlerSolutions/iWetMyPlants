Calibration Tips

  • Ensure the sensor is completely dry before setting the dry point
  • Wait for the reading to stabilize (5-10 seconds) before capturing
  • For wet point, submerge only up to the marked line on the sensor
  • The dry value should be higher than the wet value
  • Typical dry values: 2500-3500, wet values: 1000-1800
